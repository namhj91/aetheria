# Aetheria 업데이트 로그

이 파일은 버전별 변경 사항을 누적해서 기록합니다.

## Ver 0.5.7 (2026-03-21)
- 오리진 `npc_child` 선택 시 부/모가 반드시 같은 종족으로 생성되도록 수정.
- NPC 배우자 관계를 매 턴 정규화하여 단방향/유실/사망 배우자 참조가 남지 않도록 안정화.

## Ver 0.5.6 (2026-03-21)
- `be0fddf` 커밋 상태로 `assets/js/ui.js`를 롤백해 최근 UI 변경으로 발생한 버그를 되돌림.

## Ver 0.5.6 (2026-03-20)
- 건국 시 국가명이 기존 국가와 완전히 동일하면 자동으로 번호 접미를 붙여 중복을 방지하도록 개선.
- NPC 생성 시 `이름+성씨` 완전 동일 조합이 나오지 않도록 유일성 보장 로직 추가.
- PC/NPC에 10자리 고유 랜덤 시드를 부여하고, 시드 앞 2자리 차이 기반의 속궁합(100-절대차) 계산식 추가.
- 관계 시뮬레이션에 배우자 매칭(동일 종족 선호), 성관계 만족도(속궁합 기반), 낮은 확률의 임신 판정(가임기/동종족 한정) 로직 추가.
- 속궁합/임신 관련 상세 값은 디버그 모드에서만 NPC 상세창에 표시되도록 제한.

## Ver 5.5 (2026-03-19)
- 모바일 환경에서 첫 터치/클릭 제스처 시 Fullscreen API를 자동 요청하도록 추가(가능한 브라우저 한정).
- `index.html`에 모바일 웹앱/전체화면 관련 메타 태그(`viewport-fit=cover`, `mobile-web-app-capable`, `apple-mobile-web-app-capable` 등) 추가.

## Ver 5.4 (2026-03-19)
- 타이틀 화면 우하단 버전 문구를 하드코딩하지 않고 `CHANGELOG.md`의 최신 버전(첫 번째 `## Ver ...` 헤더)을 자동으로 읽어 표시하도록 변경.
- 버전 정보를 불러오는 중(`Ver Loading...`) / 실패 시 안내 문구를 출력하도록 처리.

## Ver 5.3 (2026-03-19)
- 타이틀의 버전 표기를 `Ver 5.3 Mobile UI & Changelog`로 갱신.
- 이어하기(불러오기) 모달의 모바일 가로 폭을 축소하여, 작은 화면에서 줌아웃 없이 보기 쉽도록 개선.
  - `w-[92vw]`, `sm:w-full`, `max-w-xl`, `sm:max-w-2xl`, `p-4 sm:p-6` 적용.
- `CHANGELOG.md` 파일을 추가해 버전별 업데이트 내역 누적 기록 시작.
