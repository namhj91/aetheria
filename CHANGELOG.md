# Aetheria 업데이트 로그

이 파일은 버전별 변경 사항을 누적해서 기록합니다.

버전정보: ver.0.0.11(260321-월드위협비정형화-타일정보확장)

## Ver 0.5.11 (2026-03-21)
- 이계 침식의 턴 확장 속도를 고정 1회에서 `1~3회 + 확률 조기종료` 방식으로 변경해 비정형 확산이 일어나도록 조정.
- 월드맵에 이계 침식 타일을 보라색 오버레이/테두리로 강조해 침식 구역을 즉시 식별할 수 있도록 시각화.
- 마왕성의 적대적 무리 생성 주기를 고정 주기에서 `12~28턴 랜덤`으로 변경.
- 마왕성 소속 적대적 무리의 동시 유지 수를 최대 5부대로 제한.
- 마왕성 소속 적대적 무리 이동을 목표 정착지 직진형에서 `가중치+랜덤 편차` 기반 비정형 이동으로 변경.
- 월드맵 타일 정보 모달에 `위협 정보` 항목을 추가해 이계 핵/침식 타일/마왕성/마왕군/드래곤 둥지 표시를 지원.

## Ver 0.5.10 (2026-03-21)
- `이계 침식` 월드 이벤트를 전면 재설계: 지도상의 단일 `이계 중심핵` 생성 후 턴마다 주변 타일이 단계적으로 침식되도록 변경.
- 이계 침식 타일에 대해 `원본 타일 타입`을 별도 보존하여, 핵 파괴 시 침식 타일이 원복되도록 복구 로직 추가.
- 침식 반경이 임계치에 도달하면 가장자리 타일 중 한 곳에서 `신규 이계 핵`이 분기 생성되어 연쇄 확산하도록 구현.
- 침식이 정착지 인근에 도달하면 정착지 방어력 vs 핵 전투력을 계산해, 정착지 함락 또는 핵 파괴 결과가 발생하는 전선 시스템 추가.
- `마왕 강림` 이벤트를 지속형 위협으로 개편: 지도상 마왕성 생성 후 일정 턴마다 적대적 무리 생산, 근접 정착지 침공 및 전투 판정 로직 추가.
- `드래곤의 분노` 이벤트를 둥지 기반 반복 습격으로 개편: 드래곤 둥지 생성 후 가장 가까운 정착지를 습격하고 둥지로 복귀하는 사이클 구현.
- 역사 시뮬레이션 턴 루프에 `마왕군/드래곤/이계 전선` 지속 처리 단계를 연결해 이벤트 선택 이후에도 위협이 자동 진행되도록 조정.
- 역사 시뮬레이션 중 전체 NPC가 아닌 `주요 인물`(용사/성녀/제국의 왕)만 별도 이동·행동하도록 추가.
- 주요 인물은 턴마다 목적지를 선택해 이동하며, 도착 시 전선 개입(핵/마왕군 저지), 치유/민심 안정, 내정 강화 같은 역할 효과를 발동.
- 주요 인물을 필수 고정 생성에서 `조건부 생성`으로 전환.
- `용사`는 이계/마왕/드래곤 위협이 모두 활성화된 경우에만 월드 이벤트 선택지 `용사 출현`이 등장하고, 선택 시 합류하도록 변경.
- `황제`는 특정 국가가 영토/정착지/국력 임계치를 넘길 때 제국 승격과 함께 해당 국가의 왕이 황제로 추대되도록 변경.
- `성녀`는 다수 정착지가 기근·저성장 등 위기 상태일 때 출현하도록 조정.
- 추가 중요 인물 `대마도사`, `대원수`를 도입해 각각 이계 봉인/마왕군 요격 중심의 역할 행동을 수행하도록 확장.
- 추가 중요 인물의 출현 조건을 3대 위협 이벤트(이계/마왕/드래곤)와 분리해, 문화 발전도·국가 경쟁 구도 같은 독립 조건으로 등장하도록 조정.
- `용사` 출현 후 즉시 전선 돌격하지 않고, 기존 주요 인물에게 접근해 확률/상성 기반으로 `용사단` 합류 제안을 수행한 뒤 전선 대응 단계로 전환되도록 개선.
- 세계 시뮬레이션 맵에서 `마왕성`, `이계 핵`, `마왕군 적대적 무리`, `주요 인물`의 현재 위치와 이동 경로를 오버레이 마커로 시각화.
- 역사 시뮬레이션 턴 갱신 시 맵은 재렌더만 수행하고 자동 센터 정렬(위치 재정렬)은 하지 않도록 수정.
- `드래곤의 분노`의 습격 조건을 둥지 반경 제한으로 변경해, 일정 타일 거리 내 정착지만 습격하도록 조정.
- 마왕성에서 생성되는 적대적 무리의 명칭을 무작위 현실형 부대명으로 부여하도록 개선.
